├── 📂 data/
│   ├── churn_frame.csv          ← Raw synthetic dataset (8,000 rows)
│   ├── churn_frame.parquet      ← Type-enforced Parquet (Notebook 01)
│   └── churn_scores.parquet     ← Batch scored output (Notebook 06)
│
├── 📒 notebooks/
│   ├── 01_ingest.ipynb          ← Data loading & type enforcement
│   ├── 02_eda.ipynb             ← EDA + leakage audit + time split
│   ├── 03_features.ipynb        ← Feature engineering + ColumnTransformer
│   ├── 04_baselines.ipynb       ← LogReg + RandomForest + TimeSeriesSplit
│   ├── 05_xgboost_optuna.ipynb  ← XGBoost + Optuna tuning + calibration
│   ├── 06_evaluation.ipynb      ← PR-AUC, ROC, Lift@K, SHAP, segments
│   └── 07_mlops.ipynb           ← PSI drift, Brier monitoring, deployment
│
├── 📂 models/
│   ├── preprocessor.joblib      ← Fitted ColumnTransformer
│   └── churn_calibrated.joblib  ← Calibrated XGBoost (final model)
│
├── 📂 serving/
│   └── app.py                   ← FastAPI: /score /explain /topk /health
│
├── 📂 docker/
│   └── Dockerfile               ← Container for production serving
│
├── 📂 scripts/
│   ├── generate_data.py         ← Synthetic dataset generator
│   └── build_notebooks.py       ← Notebook builder
│
└── requirements.txt             ← All Python dependencies
🧠 Model Stack
XGBoost + Optuna (30 trials)
Isotonic Probability Calibration
TimeSeriesSplit (5-fold CV)
scale_pos_weight for imbalance
PR-AUC primary metric
⚙️ Engineered Features
engagement_rate = active_days / 30
usage_per_login = hours / logins
support_intensity = tickets + 3×SLA
email_ctr = clicks / opens
price_to_tenure = billing / tenure
📡 MLOps
PSI drift (weekly, key features)
Brier score tracking over time
Retrain trigger: PSI > 0.25
Model versioned + data hash
Anonymized prediction logging
